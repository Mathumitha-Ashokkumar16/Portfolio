# Mathumitha A - Personal Portfolio

A modern, elegant, and fully responsive personal portfolio website built with React, Tailwind CSS, and Framer Motion.

## 🌟 Features

- **Modern Design**: Clean and elegant UI with soft pastel gradient theme
- **Fully Responsive**: Mobile-first design that works on all devices
- **Smooth Animations**: Beautiful animations powered by Framer Motion
- **Dark/Light Mode**: Toggle between dark and light themes
- **SEO Optimized**: Meta tags for better search engine visibility
- **Interactive Components**: Engaging user experience with hover effects and transitions

## 🛠️ Tech Stack

- **React 18** - UI library
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animation library
- **React Icons** - Icon library
- **React Type Animation** - Typewriter effect

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/Mathumitha-Ashokkumar/portfolio.git
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and visit `http://localhost:3000`

## 🚀 Build for Production

```bash
npm run build
```

The build files will be in the `dist` folder.

## 📱 Sections

- **Hero** - Introduction with typewriter animation
- **About** - Personal information and quick stats
- **Skills** - Technical skills organized by category
- **Projects** - Featured projects with descriptions
- **Certifications** - Timeline of achievements and certifications
- **Contact** - Contact form and social links
- **Footer** - Quick links and social media

## 🎨 Customization

### Colors
Edit `tailwind.config.js` to customize the color scheme:
- Primary colors (purple shades)
- Accent colors (pink shades)

### Content
Update the following files to customize content:
- `src/components/Hero.jsx` - Personal introduction
- `src/components/About.jsx` - About section
- `src/components/Skills.jsx` - Skills and technologies
- `src/components/Projects.jsx` - Project showcase
- `src/components/Certifications.jsx` - Achievements
- `src/components/Contact.jsx` - Contact information

### Profile Photo
Replace the placeholder in `src/components/About.jsx` with your actual profile photo.

## 🌐 Deployment

### Vercel
1. Push your code to GitHub
2. Import your repository on [Vercel](https://vercel.com)
3. Deploy with one click

### Netlify
1. Push your code to GitHub
2. Import your repository on [Netlify](https://netlify.com)
3. Build command: `npm run build`
4. Publish directory: `dist`
